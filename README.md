# Interactive Frontend Cart System

An interactive shopping cart built using **HTML**, **JavaScript**, and **Bootstrap** to demonstrate modern frontend development concepts through one connected application.

Rather than treating each concept as an isolated exercise, this project combines them into a realistic user interface where every component works together. Users can add products to a shopping cart, remove items dynamically, receive instant form feedback, apply promo codes, and complete a simulated checkout experience.

## Features

* Interactive shopping cart interface
* Dynamic cart item creation
* Live cart total updates
* Add and remove products
* Promo code input with instant feedback
* Checkout simulation without page refresh
* Responsive interface using Bootstrap
* Keyboard-friendly controls
* Accessible announcements for important interface updates

## Concepts Demonstrated

This project demonstrates practical frontend programming concepts, including:

* Semantic HTML
* DOM manipulation
* Dynamic element creation
* Event handling
* Event delegation
* Form handling
* Form validation
* Preventing default browser behaviour
* Custom data attributes (`data-*`)
* DOMContentLoaded initialization
* Modular JavaScript functions
* Separation of concerns

## Accessibility

Accessibility has been considered throughout the project.

Implemented accessibility features include:

* Semantic HTML5 landmarks and headings
* Keyboard-accessible controls
* ARIA live regions for dynamic announcements
* Accessible form labels
* Clear page structure for assistive technologies
* Responsive layout for improved readability

## Learning Outcomes

This project demonstrates how multiple frontend concepts work together to build a complete interactive interface rather than solving isolated exercises.

Key learning outcomes include:

* Building reusable helper functions

* Dynamically creating and updating DOM elements
* Managing application state through the DOM
